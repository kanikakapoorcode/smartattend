## 🛠 Installation
### Prerequisites
- Node.js ≥18.x
- Python ≥3.11
- PostgreSQL ≥15

```bash
# Clone with submodules
git clone --recurse-submodules https://github.com/kanikakapoorcode/smartattend.git

# Install dependencies
cd smartattend && npm install

# Set up environment
cp .env.example .env
```

### Running Locally
```bash
# Start all services
npm run dev
```
